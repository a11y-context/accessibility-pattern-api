title: Repo Install
---

## What this flow does
- Installs enforcement files into a repository
- Configures retrieval mode (MCP, docs/RAG, or hybrid)
- Ensures your AI client loads instructions and can retrieve guidance
- Verifies end-to-end behavior with a quick test

## Who it’s for
- Most teams and individual developers
- Orgs where admin-managed enforcement is unavailable or incomplete
- Any environment where portability matters

## How it works (short version)
- The kit adds **enforcement** instructions to your repo.
- When a developer makes a request, the assistant triggers **retrieval** (MCP and/or docs).
- Retrieved best practices are incorporated into AI-generated code.

## Steps (in order)
1. Choose retrieval mode
2. Install the kit
3. Configure RAG (optional, required for docs-only)
4. Enable in your client
5. Verify installation

## Start
- Go to **Choose Retrieval Mode**.
